# 任务摘要：语文页面增加好词好句、重点字词、错别字收集

**时间**: 2026-04-25 07:25 CST
**目标**: 为语文页面增加好词好句积累、重点字词学习、错别字收集功能

## 完成内容

### 1. 好词好句积累
每个单元包含：
- **5个好词**：如"浩浩荡荡、山崩地裂、人声鼎沸"等
- **2个好句**：课文中的优美句子，带引号展示

### 2. 重点字词
每个单元包含：
- **5个生字卡片**：大字展示、拼音、释义、例词
  - 悬停上浮效果
  - 宋体字体展示
- **10个词语**：标签式展示

### 3. 错别字收集
功能设计：
- **上传默写图片**：点击按钮选择图片
  - 上传后显示"正在分析..."
  - 支持自动识别（预留API）
  - 识别失败可手动输入
- **手动输入**：支持多个错别字用逗号分隔
- **记录管理**：
  - 显示错误字（红色删除线）
  - 输入正确写法
  - 显示记录日期
  - 支持删除记录

### 4. 数据内容
覆盖四年级上册8个单元：
- 40个生字（带拼音释义）
- 80个词语
- 40个好词
- 16个好句

### 5. 样式设计
- 生字卡片：网格布局，悬停动画
- 好词标签：圆角标签，绿色主题
- 好句：左侧黄色边框，斜体展示
- 错别字：红色标记，输入框可编辑

### 6. 文件修改
- `client/app.js` - 新增 GOOD_WORDS、KEY_WORDS 数据，上传/管理错别字函数
- `client/style.css` - 新增好词好句、生字卡片、错别字样式
- `client/index.html` - 版本号 v=11

## 使用说明
1. 点击语文页面的单元节点
2. 展开后可以看到：课文、古诗、好词好句、重点字词、错别字记录
3. 点击"上传默写图片"选择照片
4. 或手动输入错别字
5. 在输入框中填写正确写法
