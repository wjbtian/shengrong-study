# 🌱 永远的神的星球

> 一个为 10 岁小学生打造的个人成长记录网站

## ✨ 项目介绍

这是永远的神（四年级）的个人成长网站，用于记录学习、生活、兴趣爱好的点点滴滴。通过每日记录与正能量反馈，培养积极向上的成长心态。

**在线访问：** https://wjbtian.github.io/shengrong-study/

## 🎯 核心特色

- **现代技术栈：** Vue 3 + Vite + Pinia 响应式架构
- **高级 UI 设计：** 深色主题 + 毛玻璃效果 + 弹性动画
- **分栏布局：** 左栏（徽章 + 吉他） + 右栏（主内容）
- **高级照片墙：** 3 张卡片式设计 + 悬停动画 + 渐变光晕

## 🛠 技术栈

- **前端：** Vue 3 + Vite + Pinia + Vue Router
- **后端：** Node.js + Express + MVC 架构
- **数据库：** SQLite
- **UI 设计：** 深色主题 + 毛玻璃特效 + 弹性动画
- **样式：** CSS Grid + Flexbox + 自定义动画系统

## 🚀 本地启动

### 前端开发服务器

```bash
# 进入前端目录
cd vue-app

# 安装依赖
npm install

# 启动开发服务器（端口 5173）
npm run dev

# 访问
open http://localhost:5173
```

### 后端 API 服务器

```bash
# 进入后端目录
cd server

# 安装依赖
npm install

# 启动服务器（端口 3200）
npm start
```

### 2. 照片墙高级特效
- **方形卡片：** 1:1 比例，杂志风设计
- **悬停动画：** 上浮 8px + 弹性缩放
- **渐变光晕：** 绿紫渐变边框，悬停显现
- **毛玻璃覆盖：** backdrop-filter 模糊效果
- **文字动画：** 延迟 0.1s 滑入

### 3. 徽章系统
- 2 列整齐排列
- 解锁/锁定状态区分
- 解锁徽章发光动画
- 悬停反馈效果
