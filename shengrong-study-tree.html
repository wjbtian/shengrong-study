🌳奥数知识树 · 点击展开
  • 📊 应用题（和差/和倍/鸡兔/年龄/盈亏/植树） ▸
    • ✅ 1.1 和差问题：大数=(和+差)÷2，小数=(和-差)÷2
    • 🆕 1.2 和倍问题：1倍数=和÷（倍数+1）
    • 🆕 1.3 差倍问题：1倍数=差÷（倍数-1）
    • 🆕 1.4 鸡兔同笼·假设法：先假设全是一种，算出差来调整
    • 🆕 1.5 鸡兔同笼·抬脚法：让动物抬脚的趣味解法
    • 🆕 1.6 年龄问题：年龄差不变，年龄和随时间同增同减
    • 🆕 1.7 盈亏问题·一盈一亏：（盈+亏）÷分配差
    • 🆕 1.8 植树问题：段数=路长÷间距，端点种+1
  • 🔍 找规律（数列/周期/图形） ▸
    • 🆕 2.1 等差数列：第n项=首项+（n-1）×公差
    • 🆕 2.2 等比数列/跳级规律
    • 🆕 2.3 数量周期：总数÷周期长度=完整周期数+余数
    • 🆕 2.4 星期/日期周期（含星期几计算）
    • 🆕 2.5 图形排列规律
  • 📐 几何与面积 ▸
    • 🆕 3.1 正/长方形：周长=2×(长+宽)，面积=长×宽
    • 🆕 3.2 不规则图形：拆分法、平移法求面积
    • 🆕 3.3 数线段：点数×(点数-1)÷2
    • 🆕 3.4 数三角形/方格计数
  • 🧠 逻辑推理 / 🧩 抽屉原理 / ⚖️ 统筹 ▸
