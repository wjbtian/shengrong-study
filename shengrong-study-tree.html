🌳 英语语法知识树

  • 🔤词法▸
    • be动词：am / is / are 的用法
    • 名词单复数：a/an, -s, -es规则
    • 人称代词：I, you, he, she, it, we, they + 宾格
    • 物主代词：my, your, his, her, our, their
    • 指示代词：this, that, these, those
    • 情态动词：can（能；会）+ 动词原形
    • 动词-ing：like + V-ing
  • 📝句法▸
    • This is... / These are...
    • Can you...? Yes, I can. / No, I can't.
    • What can you see? I can see...
    • Whose is this? It's...
    • What's the matter? I'm tired/hungry.
    • 一般现在时：主语+V. / 主语+doesn't+V原形
    • 时间介词：in (morning), on (Monday), at (7 o'clock)
  • 🔊语音▸
    • 字母e发 /e/：bed, desk, pen, red, ten
    • 字母组合：oa发/əʊ/, ai发/eɪ/, ee发/iː/
    • 自然拼读：建立字母-发音对应关系

🌳 奥数知识树（点击展开）

  • 📊📊 应用题（和差倍/鸡兔/年龄/盈亏/植树）▸
    • 🟡 1.1 和差问题（大数=和+差）÷2，小数=（和-差）÷2
    • 🆕 1.2 和倍问题：1倍数=和÷（倍数+1）
    • 🆕 1.3 差倍问题：1倍数=差÷（倍数-1）
    • 🆕 1.4 鸡兔同笼·假设法：先假设全是一种，算出差来调整
    • 🆕 1.5 鸡兔同笼·抬脚法：让动物抬脚的趣味解法
    • 🆕 1.6 年龄问题：年龄差不变，年龄和随时间同增同减
    • 🆕 1.7 盈亏问题·一盈一亏：（盈+亏）÷分配差
    • 🆕 1.8 植树问题：段数=路长÷间距，端点种+1
  • 🔍🔍 找规律（数列/周期/图形）▸
    • 🆕 2.1 等差数列：第n项=首项+（n-1）×公差
    • 🆕 2.2 等比数列/跳级规律
    • 🆕 2.3 数量周期：总数÷周期长度=完整周期数+余数
    • 🆕 2.4 星期/日期周期（含星期几计算）
    • 🆕 2.5 图形排列规律
  • 📐📐 几何与面积▸
    • 🆕 3.1 正/长方形：周长=2×(长+宽)，面积=长×宽
    • 🆕 3.2 不规则图形：拆分法、平移法求面积
    • 🆕 3.3 数线段：点数×(点数-1)÷2
    • 🆕 3.4 数三角形/方格计数
  • 🧠🧠 逻辑推理 / 🧩 抽屉原理 / ⚖️ 统筹▸
    • 🆕 4.1 真假话问题：假设法，逐一排除矛盾
    • 🆕 4.2 简单排列：枚举法，有序列举
    • 🆕 4.3 抽屉原理入门：n+1只鸽，放n个笼，必有1笼≥2只
    • 🆕 5.1 时间安排：统筹规划、最短时间问题
